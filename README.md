# Advent of Code 2020

My (complete) solutions to Advent of Code 2020 in Python and some solutions in the beginning with C++.

## Python Environment

```
virtualenv -p python3.9 venv
source venv/bin/activate
pip install numpy scipy
```

## C++ Environment

* CMake
* GCC 9.3 (C++17)
* Boost 1.71.0
